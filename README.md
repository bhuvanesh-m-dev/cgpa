# 🎯 CGPA Calculator

A clean, interactive CGPA calculator with a 10-point scale, fixed course list, and shareable URL functionality. Built with pure HTML, CSS, and JavaScript, it supports O (10), A+ (9), A (8), B+ (7), B (6), C (5), and U (0) grades across 10 pre-defined courses (Ada, Aiml, Dbms, Ios, Sta, Ese, Dbms Lab, Os Lab, Quantity Apt, Power Bi) totaling 27 credits. The tool provides real-time GPA calculation, one-click reset to default 'O' grades, and generates shareable URLs that encode grade selections for easy sharing via WhatsApp, Telegram, or direct link copy. Fully responsive and browser-compatible, it's perfect for students, advisors, and self-assessment. Live demo at [bhuvanesh-m-dev.github.io/calculator](https://bhuvanesh-m-dev.github.io/calculator/). Built with Tailwind CSS and custom styling, the project is open-source under MIT License. Contributions welcome via GitHub. Made with 💙 by Bhuvanesh.
